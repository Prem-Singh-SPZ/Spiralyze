How it Works

Manage, secure, and expose all of your APIs 

  • 1. Connect an API

    Create APIs through a wizard, import existing APIs, or use a no-code API designer. Manage APIs across Gravitee, other API Gateways, and event brokers.

  • 2. Ensure security

    Expose events and APIs from brokers like Solace, Confluent, and Kafka. Apply policies to enforce security and reliability.

  • 3. Productise APIs

    Boost API discoverability with the Gravitee Developer Portal, where consumers can subscribe to APIs and explore API documentation.

Get started by building or importing your APIs

With Gravitee, you can start your API journey by creating APIs in a few different ways: you can use our creation wizard (or Management API) to create a gateway API; you can import your own APIs; or you can design an API data model with Gravitee’s no-code API designer.

How it's better:

We're able to support API types, protocols, and event brokers that competitors can't. Nobody offers the breadth of backend API and event broker support that Gravitee does, and nobody offers the flexibility of exposure and protocol mediation (see next section) as Gravitee does.
